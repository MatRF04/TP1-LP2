# Simulador de Ecosistema - TP1

**Integrante**: Mateo Rios

## Descripcion Breve
Este proyecto es una simulacion de un ecosistema en 2D, desarrollado en C++ y Qt.
El simulador modela la interraccion, reproduccion, movimiento y muerte de tres tipos de entidades vivas (Plantas, Hervivoros y Carnivoros)
esto dentro de una grilla que simula un tablero, respetando reflas de consumo de energia, envejecimiento y la capacidad de entidades que puede 
contener el ecosistema

## Compilacion y Ejecucion
1. Extraer el archivo comprimido
2. Abrir el proyecto en Qt
3. Seleccionar Open Project y elegir el archivo CMakeList.txt del proyecto
4. Hacer clic en BUILD para compilar el proyecto 
5. Hacer clicl en RUN para ejecutar el simulador
